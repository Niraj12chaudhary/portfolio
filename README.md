# Shivam Chaudhary Portfolio

Vite + React + Tailwind portfolio codebase.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Main Files

- `src/components/Home.jsx` - Hero section with profile photo and CTA
- `src/components/About.jsx` - MedShop featured case study
- `src/components/Experiance.jsx` - Full-time experience details
- `src/components/PortFolio.jsx` - Parking-Lot project reference, principles, depth, notes
- `src/components/Contact.jsx` - Contact section
